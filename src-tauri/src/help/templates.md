# Templates

## Overview

Templates let you save and restore sandbox states. Save a configured
sandbox as a template, then use it to quickly start new sessions with
the same environment.

## Operations

- **Save**: Capture a running sandbox as a reusable template
- **Load**: Import a template from a tar file
- **Remove**: Delete a saved template

## Usage

1. Start a session and configure the sandbox as needed
2. Save the sandbox as a template with a tag name
3. Use the template when creating new sessions for faster startup
