# Agents

## Built-in Agents

Beachead ships with 10 pre-configured agent types:

- **Claude Code** — Anthropic's coding agent (API key or OAuth)
- **Codex** — OpenAI's coding agent (API key or OAuth)
- **Copilot** — GitHub Copilot (GitHub token)
- **Cursor** — Cursor AI (API key or OAuth)
- **Droid** — Factory/Droid agent (API key or OAuth)
- **Gemini** — Google's coding agent (API key or interactive)
- **Kiro** — AWS Kiro agent (device flow only)
- **OpenCode** — Multi-provider agent (multiple API keys)
- **Docker Agent** — Docker's own agent (multiple API keys)
- **Shell** — Plain shell sandbox (user-configured)

## Custom Agents

You can register custom agents with a kit reference pointing to a local
kit directory or a remote kit URL.
